# CST-452 Senior Project – Secure E-Commerce Web Application
This repository contains all relevant files, documentation, and source code for my CST‑452 Senior Capstone Project, a full-stack e-commerce web application designed to simulate a real-world online shopping experience.
# Project Overview
The Secure E-Commerce Web Application enables users to:
Register and securely log in with JWT authentication.
Browse products loaded dynamically from a MongoDB database.
Add, update, or remove items in a persistent shopping cart.
Proceed through a validated checkout process and receive order confirmation.
Built using Node.js, Express.js, MongoDB Atlas, and Vanilla JavaScript, this project emphasizes secure authentication, intuitive UI/UX, and full-stack integration.
# ⚙️ Dependencies & Requirements
Please ensure all prerequisites are installed before running the project.
# Core Dependencies
Node.js (version 18+ recommended)
MongoDB Atlas (cloud-hosted database)
Node packages via npm install (see package.json)

  1) Open Terminal
  2) Type: node Server.js - Activates the database which uses mangodb atlas
  3) Navigate to Front End Page (Home.hmtl)
